# sahithlsav-software

Official software for SSF Chathamangalam Sector Sahitholsav. Includes Results Portal, Team Standings, and Fund Collection modules.

## Deployment
Deployed on Vercel. Configuration in `vercel.json`.

## Features
- Real-time Results Portal
- Team Point Table with live updates
- Digital Fund Collection with Receipt generation
- Admin Dashboard for data management
- Supabase Integration
